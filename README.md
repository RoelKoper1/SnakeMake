# SnakeMake
snakemake blok 11a

Dit Project is gemaakt om uitgevoerd te worden in een snakemake enviroment.
Voer het project uit door: "snakemake resultaten" te typen.

De resultaten zijn in resultaten.txt te vinden.

-Sequenties
-Functies
-Pubmed entries
-Pathway
